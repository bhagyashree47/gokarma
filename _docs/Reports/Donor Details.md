 ## View and download the report. 

 **To view the donor detail**: 

 1. Select donor name from the list. 

 2. Select **View Report** button. 

 **To download the report**: 

 1. Select donor name from the list. 

 2. Select **View Report** button. 

 3. Select **Download pdf** button.
